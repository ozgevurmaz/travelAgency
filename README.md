# travelMe Website

## About The Project

A tour agency website that allows user search and book easily for any tour. The website provides an easy-to-use interface for searching and booking tours.


## Technologies
⭐️ Vite Js
⭐️ TailwindCss
⭐️ Formik
⭐️ Yup
⭐️ ScrollReveal
